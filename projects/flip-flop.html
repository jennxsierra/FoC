Flip-Flop

The SR flip-flop stands for "Set-Reset" flip-flop. It has a 1-bit memory bistable device having two inputs, i.e., SET and RESET. The SET input 'S' sets the device or produces the output for 'Q' for 1, and the RESET input 'R' resets the device or produces the output for 'Q' to 0. This output depends on the set and reset conditions, which is either at the logic level "0" or "1".
